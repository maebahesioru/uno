# UNO ゲーム

React/TypeScript/Next.js/Tailwind CSSで作成されたAI対戦型UNOゲームです。

## 機能

- 🎮 プレイヤー1人 vs AI 3人の対戦
- 🎨 SVGで作成された美しいカードデザイン
- 🤖 スマートなAI対戦相手
- 🎯 完全なUNOルール実装（スキップ、リバース、ドロー2、ワイルド、ワイルドドロー4）

## セットアップ

```bash
# 依存関係のインストール
pnpm install

# 開発サーバーの起動
pnpm dev
```

ブラウザで http://localhost:3000 を開いてゲームを開始してください。

## 遊び方

1. あなたのターンになったら、手札からカードをクリックして出します
2. 捨て札の色または数字が一致するカードを出せます
3. 出せるカードがない場合は、山札をクリックしてカードを引きます
4. ワイルドカードを出すときは、次の色を選択します
5. 手札を全て出し切ったプレイヤーが勝利です

## 技術スタック

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- SVG Graphics
